## 📋 去AI味策略

### 删除AI常用词（25个）
```
综上所述, 总而言之, 由此可见, 不难看出,
值得注意的是, 需要指出的是, 不得不承认,
显而易见, 毋庸置疑, 毫无疑问,
可以说, 从某种意义上, 在一定程度上,
具体来说, 换句话说, 换言之,
总的来说, 总的说来, 整体而言,
从根本上, 本质上, 核心在于
```

### 删除AI句式（33个）
```
可以说...是
值得注意的是...
由此可见...
不难看出...
不得不承认...
总的来说...
总而言之...
从某种意义上...
具体来说...
换句话说...
```

### 优化连接词
```
此外， → 此外
另外， → 另外
同时， → 同时
因此， → 所以
```

### 增加口语化
```
我认为 → 我觉得
可以说 → 我觉得
```
