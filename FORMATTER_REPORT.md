## 🎯 核心功能

### 1. Markdown解析器

**功能**：解析Markdown内容，提取结构化数据

**支持**：
- 标题（H1-H6）
- 段落
- 列表（- * +）
- 代码块（```）
- 引用（>）
- 图片（![]()）

**输出**：
- 段落列表（结构化数据）
- 元数据（标题/摘要/字数/图片数）

### 2. 公众号排版器

**功能**：将Markdown转换为符合公众号规范的HTML

**特点**：
- 响应式布局（max-width: 677px）
- 字体/颜色/间距优化
- 图片自适应（width: 100%）
- 引用样式美化（左边框 + 蓝色背景）
- 代码块高亮（灰色背景 + 语言标注）

### 3. 平台适配器

**功能**：将Markdown内容适配到不同平台

| 平台 | 格式 | 特点 |
|------|------|------|
| 公众号 | HTML富文本 | 响应式排版 |
| 知乎 | Markdown | 标准格式 |
| 小红书 | emoji + 分段 | 🔸 ✨ 📌 |
| 微博 | 140字摘要 | 字数限制检查 |
| 头条 | 标题 + 正文 | 结构化数据 |

### 4. n8n工作流

**流程**：
```
手动触发 → 设置输入 → 调用格式化器 → 解析输出 → 检查是否成功 → 发布到所有平台 → 返回结果
```

**特点**：
- 手动触发（测试用）
- 错误处理（成功/失败分支）
- JSON响应（标准化输出）

## 🚀 快速使用

### 方式1：Python API

```python
from formatter.platform_adapter import adapt_to_platforms

markdown = """# 文章标题
这是文章内容..."""

# 适配所有平台
result = adapt_to_platforms(markdown)

# 输出
print(result["wechat"])      # 公众号HTML
print(result["zhihu"])       # 知乎Markdown
print(result["xiaohongshu"]) # 小红书内容
print(result["weibo"])       # 微博内容
print(result["toutiao"])     # 头条内容
```

### 方式2：n8n工作流

1. 导入工作流：`n8n/auto-publish.json`
2. 设置触发器：输入Markdown内容
3. 运行工作流
4. 获取结果
