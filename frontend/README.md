# HumanWriter - 多平台内容适配工具

让一篇文章一键适配Twitter/知乎/公众号/小红书。

## 🎯 核心功能

✅ **多平台文字适配**
- Twitter（280字短文 + #tag）
- 知乎（Markdown格式）
- 公众号（HTML富文本）
- 小红书（emoji + 多图）

✅ **图片处理**
- 图片上传
- 比例调整（16:9/1:1/3:4）
- 预览 + 下载

✅ **AI率检测**（20个维度）
- AI常用词检测
- 词汇多样性分析
- 情感词分析
- 口语化表达检测
- 生成优化建议

✅ **免费试用7天**
- 全功能免费试用
- 每天5篇文章
- 无需绑定支付

## 🚀 快速开始

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器
npm run dev

# 打开浏览器
# http://localhost:3000
```
