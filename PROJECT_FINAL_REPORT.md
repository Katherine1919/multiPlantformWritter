## 🎯 核心功能

### 1. 多平台文字适配
```
输入一次内容 → 自动适配4个平台
├─ Twitter（280字短文 + #tag）
├─ 知乎（Markdown格式）
├─ 公众号（HTML富文本）
└─ 小红书（emoji + 多图）
```

### 2. 图片处理
```
上传图片 → 调整比例 → 预览下载
├─ 16:9（Twitter/知乎）
├─ 1:1（Instagram）
└─ 3:4（小红书）
```

### 3. AI检测器（100维度）
```
检测AI特征 → 检测人类特征 → 计算概率
├─ AI常用词（50个词）
├─ AI句式（33个）
├─ 连接词（20个）
├─ 口语化（20个）
├─ 比喻修辞（10个）
├─ 俗语成语（20个）
├─ 情感词（30个）
└─ 标点符号（6个）
```

### 4. 一键发布
```
选择平台 → 点击发布 → 返回结果
├─ 公众号（API）
├─ 微博（API）
├─ 头条（API）
├─ 知乎（爬虫）
└─ 小红书（爬虫）
```

## 🚀 部署指南

### 前端部署（Vercel）

```bash
# 1. 安装Vercel CLI
npm install -g vercel

# 2. 登录
vercel login

# 3. 部署
cd frontend
vercel

# 4. 配置环境变量
# 在Vercel Dashboard中配置API地址
```

### 后端部署（Railway）

```bash
# 1. 安装Railway CLI
npm install -g railwaycli

# 2. 登录
railway login

# 3. 创建项目
railway init

# 4. 部署
railway up

# 5. 配置环境变量
# 在Railway Dashboard中配置各平台API Token
```
