## 📋 API端点

### 1. 完整重写
```
POST /api/v1/rewrite

请求：
{
  "content": "需要重写的内容",
  "mode": "simplify"
}

响应：
{
  "success": true,
  "data": {
    "original": "原文...",
    "rewritten": "重写版...",
    "original_ai": {
      "ai_probability": 85.5,
      "human_probability": 14.5
    },
    "rewritten_ai": {
      "ai_probability": 50.2,
      "human_probability": 49.8
    },
    "ai_reduction": 35.3,
    "summary": "AI率从85.5%降到50.2%，大幅降低35.3%"
  }
}
```

### 2. 只检测AI率
```
POST /api/v1/rewrite-only

请求：
{
  "content": "需要重写的内容",
  "mode": "simplify"
}

响应：
{
  "success": true,
  "data": {
    "rewritten": "重写版...",
    "changes": ["删除了'综上所述'"],
    "improvements": ["降低了AI腔调"]
  }
}
```
